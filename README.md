# securesocks5
一个socks5tls服务器  
It's a server.
<br>
## Support
support tls1.3
socks5tls -> proxy  
socks5tls -> socks5 -> proxy
<br>
## Dependencies
Using the following services or open-source libraries.

- [glog](https://github.com/google/glog)
- Boost.Asio or standalone Asio
